---
## ⚙️ Como rodar o projeto

```bash
# Clone do repositório
$ git clone https://github.com/nicolasteofilo/dtmoney
```

```bash
# Instalação de dependências
$ yarn

# Rodando
$ yarn start

```

Vá para [http://localhost:3000/](http://localhost:3000/) para ver o resultado 🙂
